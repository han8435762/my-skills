## 四、衍生指标计算

`calculate_indicators(df_daily)` — [`scripts/analyze.py:98`](scripts/analyze.py)

输入日K线 DataFrame，输出 `(enriched_df, summary)` 字典。

| 指标                              | 计算方式                                                  |
| --------------------------------- | --------------------------------------------------------- |
| MA5 / MA20 / MA60 / MA120 / MA250 | 收盘价滚动均值                                            |
| VOL_MA5 / VOL_MA20 / VOL_MA60     | 成交量滚动均值                                            |
| 量比\_20vs60                      | 近20日均量 / 近60日均量                                   |
| 价格百分位                        | (当前价 - 年低) / (年高 - 年低) × 100                     |
| 近3/6/12月涨跌幅                  | 当前价 vs N日前收盘价                                     |
| 近20日平均振幅                    | 振幅列均值                                                |
| 近60日最大振幅区间                | (60日最高 / 60日最低 - 1) × 100                           |
| 均线排列                          | MA5>MA20>MA60>MA120 → 多头；反之 → 空头；否则 → 缠绕/过渡 |
| 近60日MA5/MA20交叉次数            | 均线差值符号变化次数 / 2                                  |
| 近20日长上影线次数                | 上影线 > 实体 × 2 的 K 线数量                             |
| 近5日出现天量                     | 近5日最大成交量 ≥ 近1年最大成交量 × 90%                   |

## 五、四阶段判断逻辑

`judge_stage(summary, df_fund, df_lhb, cyq_summary)` — [`scripts/analyze.py:397`](scripts/analyze.py)

每个阶段独立打分，得分最高者为判断结果。

### 阶段一：建仓

核心特征：股价阴跌，成交量极低，散户绝望离场

| 条件          | 判断标准              |
| ------------- | --------------------- |
| 近1年股价下跌 | 近1年涨跌幅 < -15%    |
| 持续缩量      | 20日/60日均量比 < 0.7 |
| 换手率极低    | 近20日平均换手率 < 2% |
| 振幅小        | 近20日平均振幅 < 3%   |
| 低位区间      | 价格百分位 < 25%      |
| 年线以下      | 股价 < MA250          |

加分项：主力资金小幅净流入、龙虎榜机构净买入、获利比例低、筹码集中度收窄

### 阶段二：洗盘

核心特征：振幅加大，30% 区间震荡，成交量温和放大

| 条件       | 判断标准                   |
| ---------- | -------------------------- |
| 振幅区间   | 近60日最大振幅区间 20%~40% |
| 温和放量   | 20日/60日均量比 > 1.2      |
| 换手率温和 | 近20日平均换手率 2%~6%     |
| 不创新低   | 近20日低点 > 近1年最低点   |
| 均线缠绕   | 近60日MA5/MA20交叉 ≥ 3次   |
| 缠绕状态   | 均线排列 = 缠绕/过渡       |

加分项：获利比例 20%~50%、当前价接近平均成本（溢价 -10%~+15%）

### 阶段三：拉升

核心特征：单边上涨，成交量持续放大，散户追涨

| 条件       | 判断标准                  |
| ---------- | ------------------------- |
| 近期大涨   | 近3月涨跌幅 > 30%         |
| 多头排列   | MA5 > MA20 > MA60 > MA120 |
| 明显放量   | 20日/60日均量比 > 1.5     |
| 换手率适中 | 近20日平均换手率 3%~8%    |
| 高位区间   | 价格百分位 > 70%          |
| 频创新高   | 近20日创新高次数 ≥ 5      |
| 连续阳线   | 近5日阳线 ≥ 4根           |

加分项：主力资金大幅净流入、龙虎榜机构净买入、获利比例快速上升（>+5%）

### 阶段四：出货

核心特征：高位放量，换手率飙升，庄家悄悄离场

| 条件          | 判断标准                            |
| ------------- | ----------------------------------- |
| 高位区间      | 价格百分位 > 80%                    |
| 换手率危险 ⚠️ | 近5日平均换手率 > 10%（**权重×2**） |
| 巨量          | 20日/60日均量比 > 2.0               |
| 振幅剧烈      | 近20日平均振幅 > 5%                 |
| 高位卖压      | 近20日长上影线 ≥ 3次                |
| 前期拉升      | 近6月涨幅 > 50%                     |
| 天量 ⚠️       | 近5日出现近1年最大成交量            |

加分项：主力资金大幅净流出、龙虎榜机构大量卖出、获利比例 >80%、筹码集中度发散

### 置信度

`最高得分 / 该阶段总条件数`：≥70% → 高，≥50% → 中，<50% → 低

若第一、二名相差 ≤1 分，标注为过渡期（如"洗盘 → 出货"）。

## 六、筹码分布本地算法

`calculate_cyq_from_kline(df_daily)` + `_compute_cyq_metrics(...)` — [`scripts/analyze.py:253`](scripts/analyze.py)

**无需外部 API**，直接从已有日K线数据本地计算，与东方财富 `stock_cyq_em` 算法等价。

### 算法原理（三角分布衰减法）

```
初始：chips[0..149] = 0      # 150 个价格档位

对每个交易日 t：
  1. 旧筹码衰减
     chips[i] *= (1 - 换手率/100)

  2. 当日新筹码叠加（三角分布，顶点在均价）
     avg = (开盘 + 收盘 + 最高 + 最低) / 4
     G   = 2 / (最高 - 最低)

     price <= avg：chips[j] += (price - 最低) / (avg - 最低) × G × 换手率
     price >  avg：chips[j] += (最高 - price) / (最高 - avg) × G × 换手率

     一字板（最高 == 最低）：chips[avg_idx] += (FACTOR-1) × 换手率 / 2
```

参数：`LOOKBACK=210` 日，`FACTOR=150` 档，精度 = max(0.01, 价格范围/149)

### 输出字段

| 字段               | 含义                           | 示例（洛阳钼业） |
| ------------------ | ------------------------------ | ---------------- |
| 获利比例           | 当前价以下筹码占比             | 39.9%            |
| 平均成本           | 筹码加权均价                   | 19.7 元          |
| 成本溢价率         | (当前价 - 平均成本) / 平均成本 | -3.4%            |
| 90成本区间         | 90% 筹码分布价格范围           | 9.42 ~ 25.23 元  |
| 70成本区间         | 70% 筹码分布价格范围           | 15.18 ~ 23.90 元 |
| 90集中度           | 90% 区间集中度（越小越集中）   | 0.4563           |
| 70集中度           | 70% 区间集中度                 | 0.2231           |
| 当前价在90区间位置 | 当前价在 90% 区间内的百分位    | 60.8%            |
| 获利比例趋势       | 近10日变化量（正=获利盘增加）  | -52.4%           |
| 集中度趋势         | 近10日变化（负=筹码收窄）      | -0.0148          |

### 加分规则

| 指标         | 条件      | 加分阶段    | 含义                     |
| ------------ | --------- | ----------- | ------------------------ |
| 获利比例     | < 20%     | 建仓        | 大量套牢盘，底部吸筹特征 |
| 获利比例     | 20%~50%   | 洗盘        | 多空分歧，震荡洗盘       |
| 获利比例     | > 80%     | 出货        | 高度获利，庄家随时可撤   |
| 90集中度     | < 0.1     | 建仓 + 洗盘 | 高度集中，庄家控盘       |
| 90集中度     | > 0.3     | 出货        | 筹码分散，派发迹象       |
| 成本溢价率   | > 50%     | 出货        | 远高于成本，出货动机强   |
| 成本溢价率   | -10%~+15% | 洗盘        | 接近成本，震荡洗盘区     |
| 成本溢价率   | < -10%    | 建仓        | 跌破成本，底部特征       |
| 获利比例趋势 | > +5%     | 拉升        | 快速拉升，获利盘激增     |
| 获利比例趋势 | < -5%     | 建仓        | 股价回调，套牢盘增加     |
| 集中度趋势   | < -0.02   | 建仓        | 筹码持续收窄，庄家吸筹中 |
| 集中度趋势   | > +0.02   | 出货        | 筹码快速发散，派发迹象   |

## 七、完整分析流程

`analyze_stock(query)` — [`scripts/analyze.py:729`](scripts/analyze.py)

```
输入：股票代码或名称
  │
  ├─ 1. find_stock_code(query)           名称/代码 → 标准代码
  ├─ 2. stock_individual_info_em         获取股票简称（失败则用 stockList 名）
  ├─ 3. stock_zh_a_hist                  日K线数据（近3年，前复权）
  ├─ 4. calculate_indicators             计算所有衍生指标
  ├─ 5. stock_individual_fund_flow       主力资金流向（可选）
  ├─ 6. stock_lhb_stock_detail_em        龙虎榜席位（可选）
  ├─ 7. calculate_cyq_from_kline         本地计算筹码分布（无需网络）
  └─ 8. judge_stage                      四阶段打分 → 格式化输出
```

步骤 5、6 失败时优雅降级，不影响核心判断。
