# 泵行业字段配置指南

## 完整字段列表

### 核心业务字段（8个）

| 字段名称 | 字段类型 | 说明 | 示例值 |
|---------|---------|------|--------|
| **时间** | 日期 | 邮件发送日期（YYYY-MM-DD格式） | 2026-03-22 |
| **国家** | 单选 | 客户所在国家/地区 | 菲律宾、伊拉克、印尼 |
| **收件人** | 文本 | 客户联系人姓名 | UNIQUA CHARMAINE |
| **产品项目** | 单选 | 泵类型识别 | 稳压泵、液压手动泵 |
| **质检等级** | 单选 | 问题严重程度 | A、B、C |
| **来源** | 单选 | 邮件来源渠道 | 163邮箱、企业邮箱 |
| **第几次返修** | 数字 | 客户返修/跟踪次数 | 1、2、3... |
| **备注** | 多行文本 | 关键需求摘要 | FOR QUOTE |

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| **AI_中文** | 多行文本 | 邮件内容的实时中文翻译版本 |

## 字段映射关系说明

### 智能识别逻辑

#### 1. 国家识别
- **优先级1**：从邮件主题和正文匹配国家关键词
- **优先级2**：从发件人邮箱域名推断
- **无法识别**：标记为"未识别"

#### 2. 泵类型识别
- **优先级1**：匹配英文泵类型（更具体的优先）
- **优先级2**：匹配中文泵类型
- **无法识别**：标记为"未识别"

#### 3. 质检等级识别
- **A级**：包含严重问题关键词（损坏、不工作、泄露等）
- **B级**：包含维修问题关键词（维修、更换、性能问题等）
- **C级**：包含咨询类关键词（询价、备件、信息查询等）

#### 4. 来源识别
- 从发件人邮箱域名自动识别
- `@163.com` → 163邮箱
- `@gmail.com` → Gmail
- 包含company/corp/ltd等 → 企业邮箱

## 注意事项

1. **字段名称区分大小写**：确保字段名与代码中完全一致
2. **单选字段值**：如果代码返回的值不在可选列表中，写入会失败
3. **日期格式**：时间字段使用 `YYYY-MM-DD` 格式
4. **数字字段**：第几次返修必须是有效的数字
5. **多行文本**：备注和AI_中文字段支持长文本
